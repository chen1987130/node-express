# node-express
<br/>
启动 <br/>
启动前需要 <br/>
1.安装nodejs和npm <br/>
本地nodejs版本5.5.1,npm版本v8.9.1 <br/>
2.安装数据库mysql <br/>
数据库配置见<a href="https://github.com/chen1987130/node-express/blob/master/src/config.js"></a> <br/>
数据库脚本<a href="https://github.com/chen1987130/node-express/blob/master/doc/%E6%95%B0%E6%8D%AE%E5%BA%93%E8%84%9A%E6%9C%AC.sql"></a> <br/>
<br/>
启动顺序 <br/>
1.CMD进入node-express目录(根据自己工程目录修改) <br/>
如：D:\dev\workspace_nodejs\node-express <br/>
2.npm install <br/>
安装依赖包,只用第一次执行 <br/>
3.npm start <br/>
启动项目 <br/>
<br/>
