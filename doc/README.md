启动
1.CMD进入framework目录(根据自己工程目录修改)
如：D:\dev\workspace_nodejs\node-express
2.npm start

3.jade(模板)         文档地址:https://pugjs.org/api/getting-started.html,https://segmentfault.com/a/1190000000357534
